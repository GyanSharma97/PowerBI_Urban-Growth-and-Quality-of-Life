# Urban Infrastructure and Livability Analysis – Power BI Dashboard

This repository contains a Power BI dashboard that analyzes urban infrastructure and livability across multiple cities. The project demonstrates how business intelligence tools can be used to explore urban data, identify patterns, and generate insights that support data-driven decision making.

The dashboard focuses on several important indicators including livability, housing affordability, emissions, and transit accessibility, and compares performance across different regions.

## Project Objective

The goal of this project is to examine how infrastructure and environmental factors influence the overall livability of cities. By comparing cities with their regional peers, the analysis highlights areas where cities perform well and where improvement opportunities exist.

The analysis explores questions such as:

- Which cities perform better compared to others in their region?
- What factors are associated with improvements in livability over time?
- How do housing affordability and transit accessibility affect overall livability?
- How do infrastructure indicators differ across regions?

---

## Dashboard Features

### Gap to Peer Analysis
This section compares city performance against regional peers to highlight cities that are leading or lagging in key metrics.

### Influencer Analysis
The dashboard examines factors that influence changes in livability scores and helps identify relationships between different urban indicators.

### Regional Comparison
Cities are grouped by regions such as Midwest, Northeast, South, and West. This allows for a clearer understanding of regional differences in infrastructure and livability performance.

### Multi-Metric Analysis
The analysis includes several important metrics:

- Livability Score
- Housing Affordability
- Transit Accessibility
- Emissions

Together, these indicators provide a broader perspective on urban quality of life.

### Trend Analysis
Time-based visualizations show how city metrics change across years, helping identify trends and long-term developments.
